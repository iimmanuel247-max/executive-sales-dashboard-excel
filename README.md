# Executive Sales Dashboard

## Project Overview

This project showcases an interactive Executive Sales Dashboard developed in Microsoft Excel using a retail sales dataset. The objective was to transform raw transactional data into meaningful business insights through data cleaning, Pivot Tables, KPI cards, interactive slicers, and visual dashboards.

The dashboard enables decision-makers to monitor key business metrics, compare performance across regions and product categories, and analyze sales trends through interactive filtering.
